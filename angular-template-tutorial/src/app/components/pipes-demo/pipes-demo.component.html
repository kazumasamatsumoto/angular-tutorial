パイプ

パイプは、Angular テンプレート式で、テンプレート内のデータを宣言的に変換することを可能にする特別な演算子です。

実行結果

Purchases from {{ company | titlecase }} on {{ purchasedOn | date }}

Total: {{ amount | currency }}

同じ式で複数のパイプを組み合わせる

パイプは左から右へと順番に適用されます。最初のパイプの出力が次のパイプの入力となります。

実行結果

The event will occur on {{ scheduledOn | date | uppercase }}.

パイプにパラメータを渡す

多くのパイプはパラメータを受け取ることができます。パラメータはコロン（:）で区切ります。

実行結果

The event will occur at {{ scheduledOn | date:'hh:mm':'UTC' }}.

カスタムパイプの作成

Angularでは独自のカスタムパイプを作成することができます。以下は、文字列をケバブケース（小文字でハイフン区切り）に変換するパイプの例です。

実行結果

Original: {{ textForKebabCase }}

Kebab Case: {{ textForKebabCase | kebabCase }}

注意点

カスタムパイプを使用するには、モジュールの declarations 配列に追加するか、 スタンドアロンコンポーネントの場合は imports 配列に追加する必要があります。
